# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately rather than opening a public issue.

Email: stplotnikov@gmail.com

Please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any relevant logs, code, or proof-of-concept

I'll acknowledge reports within a few days and keep you updated as the issue is investigated and resolved.

## Supported Versions

This project follows the `main` branch as the latest supported version. Security fixes are applied there.
